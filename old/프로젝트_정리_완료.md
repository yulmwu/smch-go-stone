## 🚀 실행 방법 (간단 요약)

### Maven으로 실행 (권장)
```bash
# 의존성 설치 및 빌드
mvn clean install

# 애플리케이션 실행
mvn javafx:run
```

### Eclipse에서 실행
1. Eclipse에서 프로젝트 열기
2. Maven 프로젝트로 Import
3. F11 키로 실행
