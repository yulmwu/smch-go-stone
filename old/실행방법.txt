== GO STONE 실행 방법 ==

1. Eclipse 설정
   - Eclipse에서 __GO STONE 안에 있는 eclipse-workspace 열기
   - Package Explorer에서 Go Stone 우클릭 후 (또는 Project 클릭)
   - Properties 클릭 > Java Build Path에서 Add External JARs... 클릭
   - __GO STONE에 있는 2개의 jar 파일 추가 후 apply and close 클릭

2. JavaFX 플러그인 설치
   - Help 클릭 > Eclipse Marketplace 클릭 > Find에다가 javafx 입력 후 검색
   - e(fx)clipse 3.6.0을 install하고 > 동의하고 >
   - 설치가 다 되면 restart하겠냐는 창이 뜨는데 확인을 눌러서
   - 재시작하고

3. 프로그램 실행
   - F11을 눌러 프로그램 실행
